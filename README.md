# DSA with TypeScript

## Overview
This repository contains implementations of Data Structures and Algorithms (DSA) using TypeScript, ranging from basic arrays to advanced DSA concepts.

## Topics Covered
- Arrays
- Strings
- Linked Lists
- Stacks & Queues
- Hash Maps & Hash Tables
- Recursion
- Sorting Algorithms
- Searching Algorithms
- Trees & Binary Trees
- Graphs & Graph Algorithms
- Dynamic Programming
- Advanced DSA Topics

## Prerequisites
- Basic knowledge of TypeScript
- Node.js installed

## Getting Started
1. Clone the repository:
   ```sh
   git clone https://github.com/your-username/dsa-ts.git
   cd dsa-ts
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
3. Run TypeScript files:
   ```sh
   npx ts-node src/index.ts
   ```
